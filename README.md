# 🥗 Meal Delivery Website

A responsive landing page for a healthy meal delivery service.  
Built with pure **HTML**, **CSS**, and **JavaScript** with interactive UI components and dynamic features.

🌐 **Live Demo:** https://reflectline.github.io/meal_website

---

## ⚡ Features

- Tabbed content for meal categories
- Custom slider with navigation controls
- Calorie calculator (based on gender, weight, height, age, activity level)
- Countdown timer for promotions
- Modal windows with form submission
- Dynamic menu card rendering
- Fully responsive layout
- Clean BEM-based CSS structure

---

## 🛠 Tech Stack

- **HTML5** – Semantic markup
- **CSS3** – Styling (BEM methodology)
- **JavaScript (ES6+)** – DOM manipulation & logic
- **Webpack** – Module bundling
